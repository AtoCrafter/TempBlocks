# TempBlocks

Minecraft MOD

一時的に設置するためのブロックを追加します。


## 追加するブロック

### フレームブロック

#### 機能

設置されているフレームを右クリックすると手に持っているブロックで置き換えます。

#### レシピ

```
///
W■W
///
```

- /: 木の棒
- W: 木材

### 崩壊ブロック

#### 機能

設置後、自然消滅します。

#### レシピ

```
GGG
GFG
GGG
```

- G: ガラス
- F: 火打石

### 連鎖ブロック

#### 機能

プレイヤーによって破壊された時に隣接している連鎖ブロックを破壊します。
一続きのブロックは全て一括で破壊されます。

#### レシピ

```
FFF
S■S
FFF
```

- F: 鉄格子
- S: 糸
